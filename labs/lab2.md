**User Stories (insert below your name):**

Sam

- **As a** student **I want** to be able to see my progress in an engaging way **So that** I am encouraged to keep learning
- **As a** student **I want** to learn something in an interactive way **So that** I have fun while learning
- **As a** student **I want** lessons to presented in a format that is easy to understand **So that** the format doesn’t impact my learning
- **As an** administrator **I want** to see how many hints get used per question per user **So that** I can determine how well the curriculum is working
- **As a** student **I want** to get meaningful feedback on my responses **So that** I can learn from my mistakes
- **As an** teacher **I want** the students skill level to be tracked **So that** I can track how they are doing at a glance
- **As an** administrator **I want** the students skill level to dynamically update based on how they are doing **So that** they can be assigned questions accordingly
- **As a** student **I want** to see my lesion history **So that** I can learn from my mistakes and successes
- **As a** student **I want** to see my feedback history **So that** I can refer back to them at a later date
- **As a** student **I want** a leveling system **So that** the more I do the more rewarded I feel
- **As a** student **I want** a level leaderboard **So that** I can compete with my friends/classmates as encouragement to do more

Chinthana

- **As a** student **I want** to be able to view my ranking compared to others **so that** I can see a measurement of my performance among my peers
- **As a** student **I want** to be able to see a time estimate on how long is left in a section/chapter **so that** I can plan my schedule accordingly
- **As a** student **I want** to see the topics I excel at and struggle at **so** that I can focus on the areas that need more attention
- **As a** student **I want** to get notification reminders about tasks/material that I should focus on **so that** I can stay focused and on track.
- **As a** teacher **I want** to see the metrics of the students in a combined graph for each topic **so that** I can ensure that the students are succeeding and areas where I need to improve.
- **As a** teacher **I want** a simple and easy to understand UI **so that** I spend less time finding things and more time being productive
- **As a** student **I want** a message window where I can send messages to the teacher/system **so that** I can clarify things I’m confused about, or make requests/accommodations
- **As a** teacher **I want** a report of how much daily progress each student has done **so that** I can make sure they are on track and not getting distracted.
- **As a** student **I want** flexibility so I can work at a time that works best for me but ensure that I am still on track with my peers **so that** I won’t fall behind.

Aidan

- **As a** teacher **I want** to post announcements **so that** the students can stay informed
- **As a** student **I want** badges **so that** my hard work feels valued.
- **As a** student **I want** hints for questions **so that** I can work toward answering a question
- **As a** teacher **I want** feedback from students **so that** I can make corrections to course material
- **As a** teacher **I want** to post reminders **so that** students get informed about approaching deadlines
- **As a** teacher **I want** to post surveys **so that** I can garner feedback relating to different topics
- **As a** student **I want** to download notes **so that** I can modify them locally to better my understanding
- **As a** teacher **I want** a clean user interface **so that** students know where to find what they need
- **As a** student **I want** dark mode **so that** my eyes don’t burn in the dark
- **As a** student **I want** a calendar **so that** I can see an overview of deadlines
- **As a** student **I want** to highlight lecture notes **so that** it’s easier to review my notes

Daniel

- As a student, I want a way of asking a chatbot that can ask general questions on concepts and topics so that I can get a quick explanation to improve my understanding.
- As a student, I want a way to review all my past quiz answers so that I can learn from my mistakes
- As a teacher, I want a way to access a student’s activity so that I can monitor their progress.
- As a teacher, I want to be able to select my own plan so that I can set it up the way I want it.
- As a student, I want immediate feedback on why my answers are incorrect so that I can clear up any misconceptions.
- As a student, I want some sort of achievement feature so that I feel motivated to keep going.
- As a student, I want interactive features so that the lessons feel more engaging
- As a student, I want hints while I'm solving a problem so that I don’t get stuck
- As a teacher, I want to be able to create my own lessons so that i can teach topics my own way.

Hemanth

- **As a** student, **I want** to set personal goals/milestones **so that** i can track my progress.
- **As a** teacher **I want** to assign specific tasks based on students' skill level **so that** they can learn materials suited for their needs.
- **As a** student, **I want** personalized study recommendations based on the areas I need to improve on **so that** I can focus and study efficiently.
- **As a** student **I want** to access the app/program on mobile devices **so that** I can learn anywhere/anytime.
- **As a** student **I want** voice narration of lessons **so that** I can listen when unable to read.

Ramez

- As a student, I want a study schedule so that I can stay consistent with my learning.
- As a teacher, I want to create short quizzes within the system so that I can assess students’ understanding after each lesson.
- As a teacher, I want to receive alerts when a student consistently struggles with a topic so that I can offer extra help.
- As an administrator, I want to ensure system security with authentication controls so that user data remains protected.
